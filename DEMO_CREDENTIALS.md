# DocSpot Demo Credentials

Use these credentials to test the different user roles in the application.

## Admin User
Has access to the Admin Dashboard to manage users, doctors, and approve applications.
- **Email:** `admin@docspot.com`
- **Password:** `adminpassword`

## Doctor User
Has access to the Doctor Dashboard to manage their schedule, accept appointments, and view patient history.
- **Email:** `doctor@docspot.com`
- **Password:** `doctorpassword`

## Patient / Customer User
Has access to the Patient Portal to search for doctors, book appointments, and view their medical documents.
- **Email:** `patient@docspot.com`
- **Password:** `patientpassword`

---
*Note: These accounts are automatically re-created whenever you run `node seed.js` in the `backend` folder.*
